This web viewer displays frames processed by an Android application using OpenCV for edge detection. The implementation uses TypeScript for type safety and the HTML5 Canvas API for hardware-accelerated rendering.

The Android app captures camera frames, processes them using Canny Edge Detection in C++ via JNI, renders them with OpenGL ES 2.0, and can stream them to this web interface for visualization and debugging.
